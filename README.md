### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 图书信息
5. 好书推荐
6. 留言反馈
7. 个人中心
8. 后台管理
9. 图书预约管理
10. 退换图书管理
11. 我的收藏管理


**管理员功能：**
1. 登录
2. 个人中心
3. 学生管理
4. 图书分类管理
5. 图书信息管理
6. 图书预约管理
7. 退换图书管理
8. 管理员管理

10. 系统管理
